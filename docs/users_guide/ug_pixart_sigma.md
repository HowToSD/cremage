# How to Use PixArt-Σ (Pixart Sigma) with Cremage

## Set up
You can skip this section if you just installed Cremage. If you are upgrading your Cremage installation, please add the following lines in config.yaml:
```
pixart_sigma_ldm_model_path: None
pixart_sigma_ldm_model: None
```

## Using PixArt-Σ
Weights are automatically downloaded, so there is no need to manually download them. To generate an image using this model, just select "Pixart Sigma" in the Generator model type combobox on the Basic tab, type prompts and press Generate.

## Fine-tuned models
There are multiple types of fine-tuned models available in the community. Cremage only supports models that satisfy the following conditions:
* Weights are in a safetensors file.
* Only transformer model weights are in the file.
* The transformer model use the same weight names as the base transformer model of PixArt-Sigma.  Please read below to find out more about this requirement.

Steps:
1. Once you download a compatible model, specify the directory in the "Pixart sigma ldm model path" in File | Preferences.
2. On Basic tab, select PixArt-Sigma in the Generator model type combobox, PixArt-Sigma tab will become visible.
3. Go to this tab, you can select a compatible fine-tuned model in the Pixart sigma ldm model combobox. Cremage checks to see if the weight names are compatible. This combobox only shows the fine-tuned models that are compatible.
See Appendix below to find out the list of weight names that fine-tuned models need to contain.

## Limitations
* Only text to image flow is supported.
* Auto face fix is supported, but the custom face image that you specify is not used during auto face fix. If you want to use the custom face image, use Face Fix tool after generation.
* This feature has been only tested on Ubuntu with NVIDIA RTX 4090 (24GB VRAM). If you have less GPU VRAM, it may not work.
